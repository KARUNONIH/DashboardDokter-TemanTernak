1.2 analisis data perusahaan forbes2000

petunjuk awal: untuk menyelesaikan studi kasus ini, anda perlu memanfaatkan dataset Forbes2000, yang tersedia di paket R HSAUR. ikuti langkah berikut untuk mempersiapkan data: install paket HSAUR
install.package("HSAUR)
data("Forbes2000", package="HSAUR")

dataset ini mencakup informasi tentang 2000 perusahaan terbesar di dunia berdasarkan beberapa karakteristik, termasuk sektor, negara dan pendapatan

1.2.1 pengantar kasus

anda adalah seorang analisis data yang ditugaskan untuk mengeksplorasi dataset Forbes2000. Dataset ini mencakup informasi mengenai perusahaan global terbesar dalam berbagai sektor dan negara. Anda diminta untuk memahami distribusi Sektor industri perusahaan berdasarkan negara. Selain itu, anda juga akan memvisualisasikan data ini untuk mengidentifikasi pola-pola yang menarik.

1. Eksploarasi awal data
tampilkan struktur dataset menggunakan fungsi str() dan ringkas informasi utama menggunakan fungsi summary(). apa saja variabel yang tersedia dalam dataset ini? pilihlah dua variabel untuk dianalisis lebih lanjut.
2. menganalisis distribusi data
buat tabel kontingensi untuk menganalisis hubungan antara variabel sector dan country. gunakan fungsi table() untuk menyajikan jumlah perusahaan di setiap sektor berdasarkan negara.
3. proporsi Data
    - gunakan fungsi prop.table() untuk menghitung proporsi data dalam tabel kontingensi: berdasarkan baris (negara).
        - berdasarkan kolom (sektor)
        - untuk seluruh data
    - Jelaskan bagaimana anda menginterpretasikan proporsi yang dihasilkan.
4. visualisasi data
    - buat diagram batang yang menampilkan distribusi jumlah perusahaan di setiap sektor, menggunakan:
        - diagram batang standar tanpa argument tambahan.
        - diagram batang dengan argument beside = TRUE untuk memisahkan kategori
    - bandingkan kedua visualisasi dn interpretasikan hasilnya.
    
    Catatan: Lengkapi diagram dengan judul, grafik, label, sumbu dan legenda
